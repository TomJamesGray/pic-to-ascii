# pic-to-ascii
Converts a picture to ASCII characters
```
usage: pic-to-ascii.py [-h] [--width WIDTH] [--font_ratio FONT_RATIO]
                       [--invert]
                       picture

optional arguments:
  -h, --help            show this help message and exit
  --width WIDTH
  --font_ratio FONT_RATIO
  --invert
```
